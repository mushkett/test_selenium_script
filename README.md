
# Test shopping

This script is parsing Amazon & Best Buy for getting a best price.

## Run Locally

Clone the project

```bash
  git clone https://github.com/mushkett/test_selenium_script.git
```

Go to the project directory

```bash
cd {/specify/your/path/to/}test_selenium_script
```

Install dependencies

```bash
  pip Install requirements.txt
```

Run tests

```bash
- Mac/Linux users: `python3 -m pytest tests/test_shopping.py`
- Windows users: `py -3 -m pytest tests/test_shopping.py`  
```
